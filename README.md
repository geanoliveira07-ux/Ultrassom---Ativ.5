# Ultrassom---Ativ.5
Aqui está inserido o código em formato de biblioteca do sensor de proximidade, ou ultrassom. Além disso, conterá o código final main.c utilizado no carrinho. 
O código do Ultrassom está dentro da pista 'lib'.

(Os resultados foram registrados e compartilhados pela dupla do projeto do carrinho seguidor de linha, Gean [eu] e Henrique. Para versões anteriores e mais comprovações de testes, acessar o repositório do Henrique) 

Commit de prova
Expectativa: Fazer com que o ultrassom esteja acionando de acordo com uma certa distância
Bônus: Implementar essa lógica dentro do código main .c do carrinho. Em formato de interrupção, fazendo com que o carrinho pare após detectar certa distância.
